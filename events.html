10-12:00: Implementing an Input-Based Latin Program
This session is designed for teachers teaching Latin for the first time, or who want to transition an existing Latin program to use comprehensible input-based methodologies. This session will explain the “why” behind input-based methodologies and will model specific techniques for using Familia Romana and other resources in the classroom.
